Was ist HTML?

Welche Konventionen sollte man bei Pfaden und Dateinamen beachten?

Es gibt spezielle Zeichen, die nicht verwendet werden dürfen, weil sie im Dateisystem eine Bedeutung haben. Dazu gehören der Doppelpunkt und die Schrägstriche / und \ als Trennzeichen für Laufwerksbuchstaben und Ordner, und die Zeichen * und ? als Jokerzeichen in der Dateisuche.

Wie baut man in HTML ein Menü? 

<div class="menu">
  <a href="#" class="active">Home</a>
  <a href="#">link 1</a>
  <a href="#">Link 2</a>
  <a href="#">Link 3</a>
  <a href="#">Link 4</a>
</div>

Welche Attribute sollte man bei Bildern wie verwenden?

src attribute specifies the path (URL) to the image. alt attribute should describe the image. style,width and height attribute to specify the width and height of an image.
