9.1. SVG Grafik von Hand erstellen

Codieren Sie eine HTML-Datei mit Inline SVG, so dass sich folgendes Aussehen ergibt:

Fügen Sie als Hover-Effekt eine CSS-Animation hinzu, die die Größe der SVG-Grafik verdoppelt, wenn man mit dem Cursor über die Grafik geht.

Geben Sie hier den vollständigen HTML- und SVG-Code ein:

QUELLCODE: 

<!DOCTYPE html>
        <html>
         <body>
        
          <h1>SVG Use The Platform</h1>
        
          <svg data-v-2c77a4f5="" width="300" height="300">
              <rect data-v-2c77a4f5="" width="300" height="300" style="fill: rgb(184, 101, 101);"></rect>      
        
              <text data-v-2c77a4f5="" x="30" y="70" style="fill:White;font-weight:400;font-size:50px;font-family:Arial"> Use </text>      
        
              <line data-v-2c77a4f5="" x1="30" y1="85" x2="275" y2="85" style="stroke:White;stroke-width:4"></line>
              <text data-v-2c77a4f5="" x="30" y="165" style="fill:White;font-weight:400;font-size:50px;font-family:Arial"> The </text>
              <line data-v-2c77a4f5="" x1="30" y1="180" x2="275" y2="180" style="stroke:White;stroke-width:4"></line>      
              <text data-v-2c77a4f5="" x="30" y="265" style="fill:White;font-weight:400;font-size:50px;font-family:Arial"> Platform </text>      
                    
              <line data-v-2c77a4f5="" x1="30" y1="280" x2="275" y2="280" style="stroke:White;stroke-width:4"></line>            
          </svg>
         
         </body>
        </html>
        
        <style>
        svg:hover {
          transform: scale(2);
        }
        svg {
          transition: all 0s;
          transform-origin: 10% 0%;
        }
        </style>
